# HYROX 使用说明

项目支持网页版和桌面版。普通使用、手机摄像头和临时分享优先选择网页版；本机调试、录制视频、批量回放与模型对比选择桌面版。

## 1. 网页版

本机启动可双击 `启动网页.bat`，或在命令行运行：

```powershell
.venv\Scripts\python.exe start_web.py
```

浏览器会打开 `http://127.0.0.1:5000`。在页面中选择画面来源、动作、视角、灵敏度和识别模型，再开启摄像头或选择视频。

展开“高级设置”后，可以分别调整：

- **骨架设置**：选择完整骨架、仅上半身或仅下半身；
- **显示手指节点**：控制拇指、食指和小指姿态点，默认开启；
- **隐藏面部**：隐藏面部姿态节点，默认关闭；
- **镜像画面**：控制画面左右翻转，摄像头模式默认开启，示例和上传视频默认关闭。

“显示手指节点”“隐藏面部”和“镜像画面”是同级独立开关，不再放在“骨架设置”下拉项中。选择仅上半身或仅下半身时，未包含在该骨架范围内的节点仍不会显示。

临时公网使用双击 `启动公网访问.bat`；如需随机访问口令：

```powershell
.venv\Scripts\python.exe start_public_web.py --protected
```

公网脚本生成的 Quick Tunnel 地址每次启动都会变化。局域网中的普通 HTTP 地址通常不能让其他电脑或手机获得浏览器摄像头权限，跨设备使用应通过 HTTPS 公网地址。摄像头权限、隐私、报告下载、故障排查和部署说明见 [网页版使用说明.md](网页版使用说明.md)。

## 2. 桌面版启动

激活环境：

```powershell
.\.venv\Scripts\Activate.ps1
```

使用摄像头：

```powershell
python main.py --hyrox-action wall_ball --camera-view front
```

使用视频：

```powershell
python main.py `
  --input-video "HYROX视频\药球.mp4" `
  --hyrox-action wall_ball `
  --camera-view front `
  --no-mirror
```

## 3. 动作选择

```text
lunge
wall_ball
rowing
skierg
burpee_broad_jump
sled_push
sled_pull
farmers_carry
```

桌面版运行中按 `A` 打开动作菜单，再按 `1` 至 `8` 选择；按 `N` 循环切换；按 `V` 切换相机视角；按 `0` 关闭动作分析。

## 4. 拍摄建议

- 一次只保留一名主要运动者；
- 身体尽量完整入镜，避免脚、手和头部长期出画；
- `front` 适合观察左右对称、膝部轨迹和站姿；
- `side` 适合观察躯干角度、髋铰链、步幅和前后阶段；
- 斜视角使用 `front_left` 或 `front_right`；
- 视频输入通常使用 `--no-mirror`，摄像头预览可保留默认镜像。

视角变化会重置当前 HYROX 分析器，避免混用不同视角的阈值。视角不足以支持某项判断时会显示 `CAMERA_VIEW_LIMITED`。

## 5. 灵敏度与配置

```powershell
python main.py `
  --hyrox-action lunge `
  --hyrox-sensitivity high `
  --camera-view side
```

- `low`：更保守，减少误计数；
- `medium`：默认；
- `high`：更敏感，适合动作幅度较小或较快的视频。

自定义动作配置：

```powershell
python main.py `
  --hyrox-action rowing `
  --hyrox-config configs\hyrox\rowing.yaml `
  --camera-view side
```

## 6. 桌面版快捷键

- `Q` / `Esc`：退出；
- `A`：HYROX 动作菜单；
- `N`：下一个动作；
- `B`：切换姿态后端；
- `V`：切换相机视角；
- `S`：截图；
- `R`：标注视频录制；
- `T`：原始视频录制；
- `M`：镜像；
- `3`：通用运动学面板；
- `F`：面部关键点开关；
- `6` / `7` / `8`：无面部、上半身、下半身骨架；
- `H`：手部关键点；
- `C`：运动学会话采集。

截图和录制快捷键只适用于桌面版。网页版截图直接下载到当前设备，服务器录制已关闭。

## 7. 各动作指导范围

- **Lunge**：阶段、重复计数、步幅、躯干稳定和膝部轨迹。
- **Wall Ball**：下蹲、最低位、起身、投掷伸展与重置；不检测药球、命中或目标高度。
- **Rowing**：划船周期、腿部驱动及躯干与手臂时序；不读取功率、距离或阻力。
- **SkiErg**：拉动周期及躯干、髋部和手臂动作。
- **Burpee Broad Jump**：下地、支撑、起身、起跳、落地和动作连续性。
- **Sled Push / Pull**：推拉状态、躯干角度、步态和对称性；不检测雪橇或真实负载。
- **Farmers Carry**：行走、躯干稳定和左右对称；不检测器械重量。

## 9. 常见问题

没有识别到动作：

- 确认动作与视频内容一致；
- 确认人物完整入镜且没有明显多人干扰；
- 检查拍摄视角是否设置正确；
- 尝试提高灵敏度；
- 桌面版可使用 `--hyrox-debug` 查看姿态特征。

计数过多：

- 改用 `low` 灵敏度；
- 固定相机，减少抖动；
- 避免中途切换视角或镜像。

检查环境与完整参数：

```powershell
python -m src.doctor
python main.py --help
```
